# Pterodactyl Installer & Manager

Una herramienta automatizada en Bash para instalar, gestionar y respaldar paneles y nodos de Pterodactyl, pensada para despliegues rápidos y entornos de prueba (como Multipass).

## Requisitos
- Ubuntu 20.04, 22.04 o 24.04
- Acceso de superusuario (`root` o `sudo`)

## Instalación y Uso (Un solo comando)

Ejecuta el siguiente comando en tu terminal para lanzar el menú interactivo:

```bash
bash <(curl -sSL [https://raw.githubusercontent.com/YamiKnigth/panel-installer/main/install.sh](https://raw.githubusercontent.com/YamiKnigth/panel-installer/main/install.sh))
